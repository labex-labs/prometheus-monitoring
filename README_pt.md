Este módulo abrange Instalação do Prometheus, Monitoramento do Linux com Node Exporter, Consultas PromQL Básicas, Configuração de Alertas, Fundamentos do Alertmanager. Você dominará essas habilidades essenciais do Linux através de laboratórios práticos e desafios do mundo real.
